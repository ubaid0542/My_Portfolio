## 🌟 About The Project

Welcome to my personal 3D interactive portfolio! This project serves as a digital representation of my skills, projects, and professional background. Leveraging the power of WebGL (via Three.js and React Three Fiber) seamlessly mixed with advanced GSAP animations, the portfolio offers an engaging and ultra-modern web experience. 

### 🚀 Key Features

* **Immersive 3D Graphics**: Built using `@react-three/fiber` & `@react-three/drei` for highly performant and stunning visual interfaces.
* **Smooth Animations**: Powered by `gsap` for fluid transitions, scroll-triggered animations, and refined UI interactions.
* **Fully Responsive**: Optimized to provide a seamless experience on desktops, tablets, and mobile devices.
* **Modern UI/UX**: Custom components, dynamic styling, and sleek typography.

## 💻 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Ensure you have Node.js installed. We recommend using npm to manage dependencies.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/ubaid0542/Ubaid_Portfolio.git
   ```
2. Navigate to the project directory
   ```sh
   cd Portfolio-Website
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
